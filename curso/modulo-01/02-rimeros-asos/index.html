Herramientas Necesarias

Herramientas que Usaremos

Para seguir este curso necesitarás algunas herramientas básicas:

  1. Un editor de código: Recomendamos VS Code, Sublime Text, Atom, o similar.
  2. Git: Sistema de control de versiones.
  3. Un navegador web: Chrome, Firefox, etc.
  4. Una cuenta de GitHub.

Asegúrate de tenerlas instaladas y configuradas.

Repositorios en GitHub

¿Qué es un Repositorio?

Un repositorio (o "repo") es como una carpeta para tu proyecto. Contiene todos los archivos del proyecto (código, imágenes, documentación, etc.) y el historial completo de revisiones de cada archivo.

Conceptos clave:

  • Commit
  • Branch
  • Merge
  • Pull Request
